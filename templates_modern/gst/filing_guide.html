NIL GSTR-1 Filing

Use the "FILE NIL GSTR-1" button on the portal if you have zero sales.

  1. Login → Dashboard → GSTR-1.
  2. Check "File NIL GSTR-1" checkbox.
  3. Proceed to File with EVC.

NIL GSTR-3B Filing

One-click filing if no tax liability and no ITC claim.

  1. Login → Dashboard → GSTR-3B.
  2. Select "Yes" for "Do you want to file NIL return?".
  3. File with EVC.
